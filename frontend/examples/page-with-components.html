Ejemplo de Página con Componentes Unificados

Esta página usa header y footer desde componentes JavaScript

Ventajas de Usar Componentes

Actualización Instantánea

Cambia el footer una vez y se actualiza en todas las páginas

Menos Código

No necesitas duplicar el HTML del footer en cada página

Consistencia

Garantiza que todas las páginas tengan el mismo footer

Fácil Mantenimiento

Un solo archivo para mantener en lugar de múltiples páginas

Cómo Usar en tus Páginas

<!DOCTYPE html>
<html lang="es">
<head>
    <!-- tus meta tags y CSS -->
</head>
<body>
    <!-- Header Dinámico -->
    <div id="header-root"></div>
    
    <main>
        <!-- Tu contenido aquí -->
    </main>
    
    <!-- Footer Dinámico -->
    <div id="footer-root"></div>
    
    <!-- Scripts -->
    <script src="/js/components/header-component.js"></script>
    <script src="/js/components/footer-component.js"></script>
</body>
</html>
